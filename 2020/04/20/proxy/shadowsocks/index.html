Ubuntu shadowsocks

The client and server are both in one package which contains two Scripts.

  • Clinet : sslocal
  • Server : ssserver

In this article, I will use shadowsocks-libev to show in Ubuntu.

1. Install shadowsocks-libev

There are some things to be writed

The Python Method:

1. Install pip

sudo apt install python3-pip

2. Install shadowsock

pip3 install https://github.com/shadowsocks/shadowsocks/archive/master.zip

You can see the version using

ssserver --version

3. Config information

sudo vim /etc/shadowsocks.json

4. Open the shadowsocks

ssserver -c /etc/shadowsocks.json -d start

You can also close shadowsocks using

ssserver -c /etc/shadowsocks.json -d stop

You can check the port using

netstat -ap | grep 9898

Note :You should open the port 9898 as you write in server_port in your cloud server.

Till to now the server is ready.
Then you can config and use shadowsocks in Window, Mac, Android or IOS!
